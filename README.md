# 🎨 畅指动画 - iOS/Android 应用分发

这是一个跨平台动画演示应用的分发页面。

## 📱 支持平台
- **iOS**: 通过 .ipa 文件安装
- **Android**: 通过 .apk 文件安装

## 🌐 访问方式
- **网页版**: 直接访问 GitHub Pages 链接
- **扫码安装**: 扫描二维码快速访问

## ✨ 动画效果
- 🌊 基础波纹效果
- 🌈 彩虹色彩层
- ✨ 表情包粒子效果
- 🔄 旋转表情包效果
- 🎆 烟花闪烁效果

## 🔧 技术栈
- **Android**: Kotlin + Compose UI
- **iOS**: Kotlin Multiplatform + Compose Multiplatform
- **分发**: GitHub Pages

## 📋 安装说明
页面会自动检测设备类型并显示相应的安装选项。
